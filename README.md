# ExploreWithMe

## Реализована новая feature — Comments (комментарии к событиям)

Теперь пользователи смогут:
- добавить новый комментарий к событию;
- изменить комментарий;
- удалить комментарий;
- получить комментарии к событию;
- получить комментария по id;
- поиск комментария по тексту.


Private: закрытый API для работы с комментариями. Зарегистированные пользователи могут:
-  **POST** `/user/{{userId}}/comments` — добавить комментарий к опубликованному событию

- **PATCH** `/user/{userId}/comments//{commentId}` — измененить свой комментарий

- **DELETE** `/user/{userId}/comments//{commentId}` — удалить свой комментарий

- **GET** `/user/{userId}/comments` — получить список своих  комментариев

Admin: апи для администраторов, с помощью которого они могут:

- **GET** `/admin/comments/{commentId}` — удалить любой комментарий по его Id

Public: публичный апи для работы с комментариями:

- **GET** `/comments/event/{eventId}` — получить комментарии по Id события
